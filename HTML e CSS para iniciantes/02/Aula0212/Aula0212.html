Aprendemos as características da BOX
Content -> conteúdo das caixas
Padding -> espaço interno
Margin -> espaço externo
Border -> borda
Width -> tamanho horizontal do conteúdo, em default é setado em auto (max)
Height -> tamanho vertical da box, em default também é setado em auto.

Estilos do Browser -> maioria dos browsers tentam padronizar os estilos default em html
Reset -> CSS reset -> são padrões de código em CSS que tenta tirar todos os padrões iniciais dos browsers
Normalize -> Normalize CSS, tenta manter a conscistência dos padrões iniciais dos browsers em CSS
Reboot -> Mistura do reset com o normalize

Css INHERIT -> existem propriedades que possuem um valor padrão, por meio do inherit podemos
fazer com que propriedades de classes pai sejam herdadas por seus filhos. Ou seja, supõe-se que
exista um 'body' 'a' 'a' 'body' -> se botarmos o valor color da 'a' como inherit ele irá herdar a color do body.
