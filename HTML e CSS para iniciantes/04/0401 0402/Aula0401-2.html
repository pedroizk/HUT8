Acessibilidade

Nessa aula aprendemos a importância da marcação semântica do html
subtítulos de imagem, marcação de texto, etc...
Somente assim as pessoas com problema de visão conseguem ter a informação
do site.

Tags de acessibilidade: main -> nav -> section -> article -> aside -> footer -> header

Explicação Article x Section

Article: representa uma região do site que é autoexplicativa. Geralmente deve conter um título.

Section: utilizada para agrupar conteúdos. Porém, seu uso principal é para recortar em diversas partes os articles.

Header, Aside e Footer

Header: marca um banner, navegação principal, etc...
Aside: marca o conteúdo extra do site.
Footer: Marca o rodapé do site.
Só cria referência fora da main ou article!!!
